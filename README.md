TO DO LIST APP

A simple command line To do list application built with Python.

FEATURES
1. Add new tasks
2. Update task status
3. Delete tasks
4. Show all tasks

HOW TO RUN
"Python P3_to_do_list.py"

HOW TO USE
1. Press 1 -> Add task
2. Press 2 -> Update task status
3. Press 3 -> Delete task
4. Press 4 -> Show all tasks

BUILT WITH
- Python 3
